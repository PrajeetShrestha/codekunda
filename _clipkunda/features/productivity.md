<div class="content-section">
    <h2>Keyboard Shortcuts</h2>

    <h3>Global Hotkey</h3>
    <p>Open ClipKunda from anywhere:</p>
    <div class="code-block">
        <pre><code class="language-text">Cmd + Option + V</code></pre>
    </div>
    <p>Alternative presets available in settings (e.g., Cmd + Shift + C).</p>

    <h3>In-App Shortcuts</h3>
    <ul>
        <li><strong>Settings</strong>: <code>Cmd + ,</code></li>
        <li><strong>Quit</strong>: <code>Cmd + Q</code></li>
        <li><strong>Pin/Unpin clip</strong>: <code>Cmd + Shift + P</code> (in Detail View)</li>
    </ul>
</div>

<div class="content-section" id="transformations">
    <h2>Text Transformations</h2>
    <p>Right-click on any text clip to access powerful transformation tools. The original clip remains unchanged; the transformed text is copied to your clipboard.</p>

    <h3>Case Transformations</h3>
    <ul>
        <li><strong>UPPERCASE</strong> / <strong>lowercase</strong></li>
        <li><strong>Title Case</strong>: Capitalize first letter of each word</li>
        <li><strong>camelCase</strong> / <strong>PascalCase</strong> / <strong>snake_case</strong> / <strong>kebab-case</strong></li>
    </ul>

    <h3>Encoding/Decoding</h3>
    <ul>
        <li><strong>Base64</strong>: Encode/Decode</li>
        <li><strong>URL</strong>: Encode/Decode (percent encoding)</li>
        <li><strong>HTML</strong>: Encode/Decode entities</li>
        <li><strong>JWT</strong>: Decode JSON Web Tokens</li>
    </ul>

    <h3>Format Conversions</h3>
    <ul>
        <li><strong>JSON</strong>: Beautify, Minify, Convert to YAML/CSV</li>
        <li><strong>YAML</strong>: Convert to JSON</li>
        <li><strong>CSV</strong>: Convert to JSON arrays</li>
    </ul>

    <h3>Text Utilities</h3>
    <ul>
        <li><strong>Trim/Normalize Whitespace</strong></li>
        <li><strong>Remove Formatting</strong> (Strip markdown/HTML)</li>
        <li><strong>Sort Lines</strong> (A-Z / Z-A)</li>
        <li><strong>Remove Duplicate Lines</strong></li>
        <li><strong>Reverse Text</strong></li>
        <li><strong>Show Statistics</strong> (Counts)</li>
    </ul>
</div>
