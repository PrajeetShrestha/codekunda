<div class="content-section">
    <h2>Advanced Search</h2>
    <p>Full-text search capability across all your clips.</p>
    <ul>
        <li><strong>Scope</strong>: Searches titles, summaries, tags, OCR text from images, and original content.</li>
        <li><strong>Case-insensitive</strong>: Matches regardless of capitalization.</li>
        <li><strong>Real-time filtering</strong>: Results update instantly as you type.</li>
    </ul>
</div>

<div class="content-section">
    <h2>Tag System</h2>
    <p>Organize your clips with automatic and intelligent tagging.</p>

    <h3>AI-Generated Tags</h3>
    <p>Powered by the Local LLM, ClipKunda automatically assigns 3-6 relevant tags per clip based on content analysis.</p>

    <h3>Local Tag Detection</h3>
    <p>Intelligent detection of formats without API calls:</p>
    <ul>
        <li><strong>Structured Data</strong>: JSON, YAML, CSV, XML, HTML, SQL, JWT, Markdown.</li>
        <li><strong>Content Types</strong>: Code, URL, Email, Numbers, Text, Screenshot, Image, Color.</li>
    </ul>

    <h3>Tag Features</h3>
    <ul>
        <li><strong>Visualization</strong>: Core tags shown with bold badges; ML tags as standard chips.</li>
        <li><strong>Filtering</strong>: Filter by one or multiple tags.</li>
        <li><strong>Smart detection</strong>: Automatically identifies data format.</li>
    </ul>
</div>

<div class="content-section">
    <h2>Favorites & Pinned Clips</h2>

    <h3>Favorites</h3>
    <ul>
        <li><strong>Star items</strong>: Mark important clips.</li>
        <li><strong>Visual indicator</strong>: Yellow star icon.</li>
        <li><strong>Filter</strong>: Toggle to show only starred items.</li>
    </ul>

    <h3>Pinned Clips</h3>
    <p>Keep critical clips always visible at the top.</p>
    <ul>
        <li><strong>Priority sorting</strong>: Pinned clips always appear first.</li>
        <li><strong>Protection</strong>: Protected from auto-cleanup and bulk deletion.</li>
        <li><strong>Visuals</strong>: distinct 5% gray background and orange pin icon.</li>
        <li><strong>Shortcut</strong>: <code>Cmd + Shift + P</code> to toggle pin status.</li>
    </ul>
</div>

<div class="content-section">
    <h2>Selection Mode</h2>
    <p>Manage multiple clips efficiently.</p>
    <ul>
        <li><strong>Bulk operations</strong>: Select multiple items for batch actions.</li>
        <li><strong>Delete multiple</strong>: Remove several clips at once.</li>
        <li><strong>Selection counter</strong>: Shows number of selected items.</li>
    </ul>
</div>
