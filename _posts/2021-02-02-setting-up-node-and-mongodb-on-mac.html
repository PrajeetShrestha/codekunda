Setting up NodeJs

Setting up NodeJs is pretty straight forward in mac.

Download the installer for your respective device here and after running the installer. Go to terminal and check the installed version of the node.
node -v

If you want to go further and execute your first node application, you can use the command below in terminal. Just take a note of your current working directory. echo "console.log('hello world')" > app.js
node app.js You can see hello world printed in the terminal. What you just did is to simply create a javascript file called app.js from the terminal and execute it with nodejs.

Installing MongoDB

You would want to try to install a community edition for MongoDB first. You can get it from here
After extracting the downloaded file you would get a folder named mongodb-macOS*** rename the folder to just mongodb for ease of use.

Now go the the folder (Downloads folder normally) in which you extracted the mongodb in terminal. And type below commands: cp -a $PWD/mongodb ~ # copy the folder to the home directory
cd ~ # go to the home directory
mkdir mongodb-data # create a folder named mongodb-data, used for storing collections and other meta-data

Running mongodb server

In terminal type below commands: ~/mongodb/bin/mongod --dbpath=~/mongodb-data # starts mongodb server This initiates mongodb server and starts listening on port 27017. Don't close the tab in terminal yet or the server might stop.

Connect to mongodb server with GUI

There are various tools available for connecting to mongodb server.

  • MongoDB Compass: Download
  • Robo: Download

My Personal favorite is MongoDB Compass. After downloading and executing the installer. Open MongoDB Compass press Command + N to open a new connection. You will see UI as below: Make sure hostname is localhost and port is 27017 and just hit connect. You will see default collections already initialized like admin, config etc. Ignore it for now.
You are all set for first CRUD application in MongoDB with NodeJs. But, That will be in second part of this tutorial.
