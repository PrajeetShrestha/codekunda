Overview

ClipKunda is a privacy-first clipboard manager for macOS that elevates everyday workflows with on-device intelligence. It captures text, images, colors, and files, then organizes, analyzes, and augments your data without leaving your machine. Built with SwiftUI and SwiftData, it delivers native performance and a refined experience with OCR, color tools, rule automation, and optional local LLM chat powered by MLX.

Technical Specifications

Supported Platforms

  • macOS 14.0+ (Sonoma, Sequoia, or later)

System Requirements

  • Apple Silicon or Intel mac with macOS 14+
  • 200MB free disk space for app data
  • Optional: MLX-compatible local models for LLM chat

Key Enhancements

  • High-accuracy OCR with image metadata extraction
  • Advanced Color Studio with palette extraction and conversions
  • Rule Engine for automated tagging and actions
  • On-device LLM chat (opt-in, no network calls)
  • Improved search, tagging, and organization capabilities
  • Transparent pixel filtering and smart caching for images
