Search & Organization

Advanced Search

Full-text search capability across all your clips.

  • Scope: Searches titles, summaries, tags, OCR text from images, and original content.
  • Case-insensitive: Matches regardless of capitalization.
  • Real-time filtering: Results update instantly as you type.

Tag System

Organize your clips with automatic and intelligent tagging.

AI-Generated Tags

Powered by the Local LLM, ClipKunda automatically assigns 3-6 relevant tags per clip based on content analysis.

Local Tag Detection

Intelligent detection of formats without API calls:

  • Structured Data: JSON, YAML, CSV, XML, HTML, SQL, JWT, Markdown.
  • Content Types: Code, URL, Email, Numbers, Text, Screenshot, Image, Color.

Tag Features

  • Visualization: Core tags shown with bold badges; ML tags as standard chips.
  • Filtering: Filter by one or multiple tags.
  • Smart detection: Automatically identifies data format.

Favorites & Pinned Clips

Favorites

  • Star items: Mark important clips.
  • Visual indicator: Yellow star icon.
  • Filter: Toggle to show only starred items.

Pinned Clips

Keep critical clips always visible at the top.

  • Priority sorting: Pinned clips always appear first.
  • Protection: Protected from auto-cleanup and bulk deletion.
  • Visuals: distinct 5% gray background and orange pin icon.
  • Shortcut: Cmd + Shift + P to toggle pin status.

Selection Mode

Manage multiple clips efficiently.

  • Bulk operations: Select multiple items for batch actions.
  • Delete multiple: Remove several clips at once.
  • Selection counter: Shows number of selected items.
