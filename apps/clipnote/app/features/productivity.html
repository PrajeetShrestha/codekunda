Productivity Shortcuts & Tools

Productivity Tools

Keyboard Shortcuts

Global Hotkey

Open ClipKunda from anywhere:

Cmd + Option + V

Alternative presets available in settings (e.g., Cmd + Shift + C).

In-App Shortcuts

  • Settings: Cmd + ,
  • Quit: Cmd + Q
  • Pin/Unpin clip: Cmd + Shift + P (in Detail View)

Text Transformations

Right-click on any text clip to access powerful transformation tools. The original clip remains unchanged; the transformed text is copied to your clipboard.

Case Transformations

  • UPPERCASE / lowercase
  • Title Case: Capitalize first letter of each word
  • camelCase / PascalCase / snake_case / kebab-case

Encoding/Decoding

  • Base64: Encode/Decode
  • URL: Encode/Decode (percent encoding)
  • HTML: Encode/Decode entities
  • JWT: Decode JSON Web Tokens

Format Conversions

  • JSON: Beautify, Minify, Convert to YAML/CSV
  • YAML: Convert to JSON
  • CSV: Convert to JSON arrays

Text Utilities

  • Trim/Normalize Whitespace
  • Remove Formatting (Strip markdown/HTML)
  • Sort Lines (A-Z / Z-A)
  • Remove Duplicate Lines
  • Reverse Text
  • Show Statistics (Counts)

Collaboration & Sharing

  • Multi-select export: Export bundles as Markdown, HTML, or PDF.
  • Share flows: AirDrop, Messages, Mail support.
  • Link handoff: Local share link for same-network discovery.
  • Privacy controls: Redact sensitive fields on export.
